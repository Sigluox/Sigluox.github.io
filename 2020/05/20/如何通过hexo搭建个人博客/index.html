##环境配置
###Node
####什么是Node.js？
#####Node是Hexo的构建引擎，用来实现博客界面的渲染。

####Node的下载安装配置
###Git
####什么是Git？
#####GIt是Hexo的版本控制工具，负责博客与服务器的上传和下载。
####为什么用Git？
####Git的下载安装配置
###Hexo
####什么是Hexo？
####Hexo的安装配置
#####首先我们先检测一下Node和Git的版本，确保环境没有问题，打开Git-bash，键入：
$node -v
$git –version
#####如果你的环境没有问题，那么我们可以开始安装Hexo，键入：
$npm install hexo-cli -g
#####安装完成后，新建一个文件夹作为你博客的根目录，在该文件夹右键Git-bash，键入:
$hexo init blog
$cd blog
$npm install
$hexo server
#####无意外情况会输出如下结果：
$INFO Start processing
$INFO Hexo is running at http://localhost:4000/. Press Ctrl+C to stop.
####Hexo的常用指令
$hexo generate #构建博客
$hexo server #上传到本地服务器，地址为http://localhost:4000/
$hexo deloyer #上传到网络服务器，地址一般为[username].github.io/[blogname].github.io
